*** Polling

- Since you‚Äôre already on React Query, using setInterval in the component is brittle (cleanup, double intervals, stale closures, multiple tabs). React Query can poll for you via query options, tied to tab visibility.
- Concept: make the game query refetch itself every N ms while the game is ongoing. Stop when status is winner/draw, or when the component unmounts.
- Sanity checks:
  - What interval feels responsive but not spammy (e.g., 1‚Äì2s)?
  - What happens in two tabs if both poll and both try to move at once? Where is ‚Äúturn validity‚Äù enforced?
